# Moose Jobs

Moose Jobs is a local job marketplace platform designed to connect employers 
with service providers for short-term tasks in small communities.

## Features
- User authentication with role-based access (employer / worker)
- Job posting and management
- Bidding and job assignment workflow
- Job completion tracking

## Tech Stack
- PHP (Laravel)
- MySQL
- RESTful API
- Tailwind CSS

## Status
This project is under active development.

## Future Improvements
- Online payments
- Review and rating system
- Notifications
